Para conectarnos a través del protocolo telnet es necesario preparar nuestra máquina virtual o servidor linux es necesario:


*  conocer la dir ip de nuestra VM o server linux 
*  tener en cuenta el user y contraseña
*  instalar el paquete ssh (telnet)






*Por defecto el servicio ya está activo, si no es necesario activarlo a través de comandos adjuntados en la doc$- Pc remoto
Es necesario habilitar el cliente telnet ya sea a gráficamente o en línea de comandos cmd administrador




Una vez completada la preparación; ponemos el siguiente comando en cmd:
 telnet [ip del servidor o VM]
*posteriormente nos pedirá un usuario y contraseña
Una vez verificado las credenciales ya estaremos dentro de nuestro server linux o VM y podremos acceder al shell de la $***
